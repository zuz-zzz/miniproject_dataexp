Biomedical Engineering - Miniproject: Data explotarion¶

In this project data from The Global Terrorism Database (GTD) - open-source database including information on terrorist attacks around the world from 1970 through 2017, was used. The task was to think of 5 questions one could have after seeing the data.

Loading and reparing the data¶

In the first step all needed libraries were imported and data was loaded using pandas read_csv function.

In [1]:
import numpy as np
import pandas as pd
import csv
import pycountry
import plotly.plotly as py
import plotly.graph_objs as go
from plotly.offline import download_plotlyjs, init_notebook_mode, plot, iplot
import matplotlib.pyplot as plt
import seaborn as sns

terror = pd.read_csv('globalterrorismdb_0718dist.csv', encoding='ISO-8859-1', low_memory=False)
terror['casualties'] = terror['nkill'] + terror['nwound']
terror.rename(columns = {'iyear': 'year', 'targtype1_txt' : 'target', 'attacktype1_txt' : 'attack_type', 'weaptype1_txt' : 'weapon'}, inplace = True)
terror = terror[['year','country', 'country_txt', 'casualties','target', 'attack_type', 'weapon', 'gname']]

1. TOP10 countries with the highest number of casualties¶

To have a quick overlook on what parts of the world are mostly affected by terrorist attacks a map was created. Only 10 countries with highest number of casualties (number of people killed, but also those who were wounded) were taken into consideration and showed on the map. To create the map using plotly, country codes were needed. They were imported from pycountry library and assigned to countries' names.

In [2]:
highest_terror = terror.groupby('country_txt')['casualties'].sum().sort_values(ascending=False).head(10)
df = highest_terror.reset_index()
input_countries = df['country_txt']

countries = {}
for country in pycountry.countries:
    countries[country.name] = country.alpha_3

codes = [countries.get(country, 'SYR') for country in input_countries]
df['codes'] = codes

data = dict(
        type = 'choropleth',
        locations = df['codes'],
        z = df['casualties'],
        text = df['country_txt'],
        colorbar = {'title' : 'Countries and casualities'},
      ) 
layout = dict(
    title = 'Countries and casualities',
    geo = dict(
        showframe = False,
        projection = {'type':'mercator'}
    )
)

init_notebook_mode(connected=True) 

choromap = go.Figure(data = [data],layout = layout)
iplot(choromap)
